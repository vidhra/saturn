# delete-sdi-source

## Description

Delete an SdiSource. The SdiSource must not be part of any SidSourceMapping and must not be attached to any input.

See also: [AWS API Documentation](https://docs.aws.amazon.com/goto/WebAPI/medialive-2017-10-14/DeleteSdiSource)
