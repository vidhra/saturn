# create-local-gateway-route-table-vpc-association

## Description

Associates the specified VPC with the specified local gateway route table.

See also: [AWS API Documentation](https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/CreateLocalGatewayRouteTableVpcAssociation)
