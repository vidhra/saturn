# mailmanager

## Description

The Amazon SES Mail Manager API contains operations and data types that comprise the Mail Manager feature of [Amazon Simple Email Service (SES)](http://aws.amazon.com/ses) .

Mail Manager is a set of Amazon SES email gateway features designed to help you strengthen your organizationâs email infrastructure, simplify email workflow management, and streamline email compliance control. To learn more, see the [Mail Manager chapter](https://docs.aws.amazon.com/ses/latest/dg/eb.html) in the *Amazon SES Developer Guide* .
