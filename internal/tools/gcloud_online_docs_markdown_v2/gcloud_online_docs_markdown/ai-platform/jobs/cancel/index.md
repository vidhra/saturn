**NAME**

: **gcloud ai-platform jobs cancel - cancel a running AI Platform job**

**SYNOPSIS**

: **`gcloud ai-platform jobs cancel` `[JOB](https://cloud.google.com/sdk/gcloud/reference/ai-platform/jobs/cancel#JOB)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/ai-platform/jobs/cancel#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `gcloud ai-platform jobs cancel` cancels a running AI Platform job.
If the job is already finished, the command will not perform an operation and
exit successfully.

**POSITIONAL ARGUMENTS**

: **`JOB`**:
Name of the job.
