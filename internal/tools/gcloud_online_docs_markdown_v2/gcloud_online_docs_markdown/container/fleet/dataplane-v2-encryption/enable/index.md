**NAME**

: **gcloud container fleet dataplane-v2-encryption enable - enables the Dataplane V2 Encryption Feature**

**SYNOPSIS**

: **`gcloud container fleet dataplane-v2-encryption enable` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/container/fleet/dataplane-v2-encryption/enable#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: This command enables the Dataplane V2 Encryption Feature in a fleet.

**EXAMPLES**

: To enable the Dataplane V2 Encryption Feature, run:

```
gcloud container fleet dataplane-v2-encryption enable
```
