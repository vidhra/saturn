**NAME**

: **gcloud app versions describe - display all data about an existing version**

**SYNOPSIS**

: **`gcloud app versions describe` `[VERSION](https://cloud.google.com/sdk/gcloud/reference/app/versions/describe#VERSION)` `[--service](https://cloud.google.com/sdk/gcloud/reference/app/versions/describe#--service)`=`SERVICE`, `[-s](https://cloud.google.com/sdk/gcloud/reference/app/versions/describe#-s)` `[SERVICE](https://cloud.google.com/sdk/gcloud/reference/app/versions/describe#SERVICE)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/app/versions/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Display all data about an existing version.

**POSITIONAL ARGUMENTS**

: **`VERSION`**:
The ID of the version to show.

**REQUIRED FLAGS**

: **--service**:
The service corresponding to the version to show.
