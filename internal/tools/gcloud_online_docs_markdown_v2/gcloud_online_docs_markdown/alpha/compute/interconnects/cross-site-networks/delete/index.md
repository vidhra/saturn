**NAME**

: **gcloud alpha compute interconnects cross-site-networks delete - delete Compute Engine cross site networks**

**SYNOPSIS**

**DESCRIPTION**

: `(ALPHA)` `gcloud alpha compute interconnects
cross-site-networks delete` deletes Compute Engine cross site networks.

**POSITIONAL ARGUMENTS**

: **`NAME` [`NAME` …]**:
Names of the crossSiteNetworks to delete.
