**NAME**

: **gcloud compute url-maps describe - describe a URL map**

**SYNOPSIS**

: **`gcloud compute url-maps describe` `[URL_MAP](https://cloud.google.com/sdk/gcloud/reference/compute/url-maps/describe#URL_MAP)` [`[--global](https://cloud.google.com/sdk/gcloud/reference/compute/url-maps/describe#--global)`     | `[--region](https://cloud.google.com/sdk/gcloud/reference/compute/url-maps/describe#--region)`=`REGION`] [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/compute/url-maps/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `gcloud compute url-maps describe` displays all data associated with
a URL map in a project.

**POSITIONAL ARGUMENTS**

: **`URL_MAP`**:
Name of the URL map to describe.

**FLAGS**

: **--global**
