**NAME**

: **gcloud compute disk-types describe - describe a Compute Engine disk type**

**SYNOPSIS**

: **`gcloud compute disk-types describe` `[DISK_TYPE](https://cloud.google.com/sdk/gcloud/reference/compute/disk-types/describe#DISK_TYPE)` [`[--zone](https://cloud.google.com/sdk/gcloud/reference/compute/disk-types/describe#--zone)`=`ZONE`] [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/compute/disk-types/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `gcloud compute disk-types describe` displays all data associated
with a Compute Engine disk type.

**POSITIONAL ARGUMENTS**

: **`DISK_TYPE`**:
Name of the disk type to describe.

**FLAGS**

: **--zone**:
Zone of the disk type to describe. Overrides the default
`compute/zone` property value for this command invocation.
