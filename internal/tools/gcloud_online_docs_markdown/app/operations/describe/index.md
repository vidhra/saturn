**NAME**

: **gcloud app operations describe - describes the operation**

**SYNOPSIS**

: **`gcloud app operations describe` `[OPERATION](https://cloud.google.com/sdk/gcloud/reference/app/operations/describe#OPERATION)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/app/operations/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Describes the operation.

**EXAMPLES**

: To describe an App Engine operation called o1, run:

```
gcloud app operations describe o1
```

**POSITIONAL ARGUMENTS**

: **`OPERATION`**:
ID of operation.
