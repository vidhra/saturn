**NAME**

: **gcloud app services describe - display all data about an existing service**

**SYNOPSIS**

: **`gcloud app services describe` `[SERVICE](https://cloud.google.com/sdk/gcloud/reference/app/services/describe#SERVICE)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/app/services/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Display all data about an existing service.

**EXAMPLES**

: To show all the data about service s1, run

```
gcloud app services describe s1
```

**POSITIONAL ARGUMENTS**

: **`SERVICE`**:
The service to describe.
