**NAME**

: **gcloud survey - invoke a customer satisfaction survey for Google Cloud CLI**

**SYNOPSIS**

: **`gcloud survey` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/survey#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: To permanently disable the survey prompt, run:

```
gcloud config set survey/disable_prompts True
```

**EXAMPLES**

: To launch the survey, run:

```
gcloud survey
```
