**NAME**

: **gcloud firebase test android locales describe - describe an Android locale**

**SYNOPSIS**

: **`gcloud firebase test android locales describe` `[LOCALE](https://cloud.google.com/sdk/gcloud/reference/firebase/test/android/locales/describe#LOCALE)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/firebase/test/android/locales/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Describe an Android locale.

**EXAMPLES**

: To see the attributes of the Android locale 'my-locale', run:

```
gcloud firebase test android locales describe my-locale
```

**POSITIONAL ARGUMENTS**

: **`LOCALE`**:
The locale to describe, found using $ [gcloud firebase
test android locales list](https://cloud.google.com/sdk/gcloud/reference/firebase/test/android/locales/list).
