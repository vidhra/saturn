**NAME**

: **gcloud firebase test android models describe - describe an Android model**

**SYNOPSIS**

: **`gcloud firebase test android models describe` `[MODEL_ID](https://cloud.google.com/sdk/gcloud/reference/firebase/test/android/models/describe#MODEL_ID)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/firebase/test/android/models/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Describe an Android model.

**EXAMPLES**

: To see the attributes of the android model 'my-model', run:

```
gcloud firebase test android models describe my-model
```

**POSITIONAL ARGUMENTS**

: **`MODEL_ID`**:
ID of the model to describe, found using $ [gcloud firebase
test android models list](https://cloud.google.com/sdk/gcloud/reference/firebase/test/android/models/list).
