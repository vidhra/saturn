**NAME**

: **gcloud transfer agent-pools describe - get details about a specific agent pool**

**SYNOPSIS**

: **`gcloud transfer agent-pools describe` `[NAME](https://cloud.google.com/sdk/gcloud/reference/transfer/agent-pools/describe#NAME)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/transfer/agent-pools/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Get details about a specific agent pool.

**EXAMPLES**

: To monitor an agent pool, run:

```
gcloud transfer agent-pools describe NAME
```

**POSITIONAL ARGUMENTS**

: **`NAME`**:
The name of the agent pool you want to describe.
