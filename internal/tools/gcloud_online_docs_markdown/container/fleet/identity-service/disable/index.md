**NAME**

: **gcloud container fleet identity-service disable - disable Identity Service Feature**

**SYNOPSIS**

: **`gcloud container fleet identity-service disable` [`[--force](https://cloud.google.com/sdk/gcloud/reference/container/fleet/identity-service/disable#--force)`] [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/container/fleet/identity-service/disable#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: This command disables the Identity Service Feature in a fleet.

**EXAMPLES**

: To disable the Identity Service Feature, run:

```
gcloud container fleet identity-service disable
```

**FLAGS**

: **--force**:
Disable this feature, even if it is currently in use. Force disablement may
result in unexpected behavior.
