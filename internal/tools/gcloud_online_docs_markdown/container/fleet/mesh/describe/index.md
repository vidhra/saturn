**NAME**

: **gcloud container fleet mesh describe - describe the status of Service Mesh Feature resource**

**SYNOPSIS**

: **`gcloud container fleet mesh describe` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/container/fleet/mesh/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Describe the status of Service Mesh Feature resource in a fleet.

**EXAMPLES**

: To describe the Service Mesh Feature, run:

```
gcloud container fleet mesh describe
```
