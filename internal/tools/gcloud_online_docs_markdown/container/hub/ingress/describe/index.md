**NAME**

: **gcloud container hub ingress describe - describe the status of Multi-cluster Ingress Feature resource**

**SYNOPSIS**

: **`gcloud container hub ingress describe` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/container/hub/ingress/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: This command describes the status of Multi-cluster Ingress Feature resource in
fleet.

**EXAMPLES**

: To describe the Ingress Feature, run:

```
gcloud container hub ingress describe
```
