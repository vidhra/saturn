**NAME**

: **gcloud scheduler locations describe - show details about a location**

**SYNOPSIS**

: **`gcloud scheduler locations describe` `[LOCATION](https://cloud.google.com/sdk/gcloud/reference/scheduler/locations/describe#LOCATION)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/scheduler/locations/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Show details about a location.

**EXAMPLES**

: To describe a location:

```
gcloud scheduler locations describe my-location
```

**POSITIONAL ARGUMENTS**

: **`LOCATION`**:
The Cloud location to describe.
