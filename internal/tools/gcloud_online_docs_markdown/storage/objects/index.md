# gcloud storage objects  |  Google Cloud CLI Documentation

*Source: [https://cloud.google.com/sdk/gcloud/reference/storage/objects](https://cloud.google.com/sdk/gcloud/reference/storage/objects)*

**NAME**

: **gcloud storage objects - manage Cloud Storage objects**

**SYNOPSIS**

: **`gcloud storage objects` `[COMMAND](https://cloud.google.com/sdk/gcloud/reference/storage/objects#COMMAND)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/storage/objects#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Manage Cloud Storage objects.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**COMMANDS**

: ``COMMAND`` is one of the following:

**`[compose](https://cloud.google.com/sdk/gcloud/reference/storage/objects/compose)`**:
Concatenate a sequence of objects into a new composite object.

**`[describe](https://cloud.google.com/sdk/gcloud/reference/storage/objects/describe)`**:
Describe a Cloud Storage object.

**`[list](https://cloud.google.com/sdk/gcloud/reference/storage/objects/list)`**:
Lists Cloud Storage objects.

**`[update](https://cloud.google.com/sdk/gcloud/reference/storage/objects/update)`**:
Update Cloud Storage objects.

**NOTES**

: This variant is also available:

```
gcloud alpha storage objects
```
