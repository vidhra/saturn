**NAME**

: **gcloud service-directory locations list - lists locations**

**SYNOPSIS**

: **`gcloud service-directory locations list` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/service-directory/locations/list#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Lists locations.

**EXAMPLES**

: To describe a Service Directory location, run:

```
gcloud service-directory locations list
```
