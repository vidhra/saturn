**NAME**

: **gcloud compute backend-buckets delete - delete backend buckets**

**SYNOPSIS**

: **`gcloud compute backend-buckets delete` `[BACKEND_BUCKET_NAME](https://cloud.google.com/sdk/gcloud/reference/compute/backend-buckets/delete#BACKEND_BUCKET_NAME)` [`[BACKEND_BUCKET_NAME](https://cloud.google.com/sdk/gcloud/reference/compute/backend-buckets/delete#BACKEND_BUCKET_NAME)` …] [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/compute/backend-buckets/delete#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `gcloud compute backend-buckets delete` deletes one or more backend
buckets.

**POSITIONAL ARGUMENTS**

: **`BACKEND_BUCKET_NAME` [`BACKEND_BUCKET_NAME` …]**:
Names of the backend buckets to delete.
