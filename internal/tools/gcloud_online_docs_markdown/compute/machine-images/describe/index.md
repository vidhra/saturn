**NAME**

: **gcloud compute machine-images describe - describe a Compute Engine machine image**

**SYNOPSIS**

: **`gcloud compute machine-images describe` `[IMAGE](https://cloud.google.com/sdk/gcloud/reference/compute/machine-images/describe#IMAGE)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/compute/machine-images/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Describe a Compute Engine machine image.

**EXAMPLES**

: To describe a machine image, run:

```
gcloud compute machine-images describe my-machine-image
```

**POSITIONAL ARGUMENTS**

: **`IMAGE`**:
Name of the machineImage to describe.
