**NAME**

: **gcloud compute http-health-checks delete - delete HTTP health checks**

**SYNOPSIS**

: **`gcloud compute http-health-checks delete` `[NAME](https://cloud.google.com/sdk/gcloud/reference/compute/http-health-checks/delete#NAME)` [`[NAME](https://cloud.google.com/sdk/gcloud/reference/compute/http-health-checks/delete#NAME)` …] [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/compute/http-health-checks/delete#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `gcloud compute http-health-checks delete` deletes one or more
Compute Engine HTTP health checks.

**POSITIONAL ARGUMENTS**

: **`NAME` [`NAME` …]**:
Names of the HTTP health checks to delete.
