**NAME**

: **gcloud pubsub message-transforms validate - validates a message transform**

**SYNOPSIS**

: **`gcloud pubsub message-transforms validate` `[--message-transform-file](https://cloud.google.com/sdk/gcloud/reference/pubsub/message-transforms/validate#--message-transform-file)`=`MESSAGE_TRANSFORM_FILE` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/pubsub/message-transforms/validate#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Validates a message transform.

**REQUIRED FLAGS**

: **--message-transform-file**:
Path to YAML or JSON file containing a message transform.
