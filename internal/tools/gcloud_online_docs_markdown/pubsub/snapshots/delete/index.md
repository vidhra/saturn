**NAME**

: **gcloud pubsub snapshots delete - deletes one or more Cloud Pub/Sub snapshots**

**SYNOPSIS**

: **`gcloud pubsub snapshots delete` `[SNAPSHOT](https://cloud.google.com/sdk/gcloud/reference/pubsub/snapshots/delete#SNAPSHOT)` [`[SNAPSHOT](https://cloud.google.com/sdk/gcloud/reference/pubsub/snapshots/delete#SNAPSHOT)` …] [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/pubsub/snapshots/delete#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Deletes one or more Cloud Pub/Sub snapshots.

**POSITIONAL ARGUMENTS**

: **`SNAPSHOT` [`SNAPSHOT` …]**:
One or more snapshot names to delete.
