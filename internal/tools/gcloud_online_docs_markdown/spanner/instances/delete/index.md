**NAME**

: **gcloud spanner instances delete - delete a Cloud Spanner instance**

**SYNOPSIS**

: **`gcloud spanner instances delete` `[INSTANCE](https://cloud.google.com/sdk/gcloud/reference/spanner/instances/delete#INSTANCE)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/spanner/instances/delete#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Delete a Cloud Spanner instance.

**EXAMPLES**

: To delete a Cloud Spanner instance, run:

```
gcloud spanner instances delete my-instance-id
```

**POSITIONAL ARGUMENTS**

: **`INSTANCE`**:
Cloud Spanner instance ID.
