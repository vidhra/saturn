**NAME**

: **gcloud sql instances describe - displays configuration and metadata about a Cloud SQL instance**

**SYNOPSIS**

: **`gcloud sql instances describe` `[INSTANCE](https://cloud.google.com/sdk/gcloud/reference/sql/instances/describe#INSTANCE)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/sql/instances/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Displays configuration and metadata about a Cloud SQL instance.
Information such as instance name, IP address, region, the CA certificate and
configuration settings will be displayed.

**POSITIONAL ARGUMENTS**

: **`INSTANCE`**:
Cloud SQL instance ID.
