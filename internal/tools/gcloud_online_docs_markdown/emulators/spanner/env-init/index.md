**NAME**

: **gcloud emulators spanner env-init - print the commands required to export Spanner emulator's env variables**

**SYNOPSIS**

: **`gcloud emulators spanner env-init` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/emulators/spanner/env-init#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Print the commands required to export Spanner emulator's env variables.

**EXAMPLES**

: To print the env variables exports for a Spanner emulator, run:

```
gcloud emulators spanner env-init
```
