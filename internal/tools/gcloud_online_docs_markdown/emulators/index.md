# gcloud emulators  |  Google Cloud CLI Documentation

*Source: [https://cloud.google.com/sdk/gcloud/reference/emulators](https://cloud.google.com/sdk/gcloud/reference/emulators)*

**NAME**

: **gcloud emulators - set up your local development environment using emulators**

**SYNOPSIS**

: **`gcloud emulators` `[GROUP](https://cloud.google.com/sdk/gcloud/reference/emulators#GROUP)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/emulators#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Set up your local development environment using emulators.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**GROUPS**

: ``GROUP`` is one of the following:

**`[firestore](https://cloud.google.com/sdk/gcloud/reference/emulators/firestore)`**:
Manage your local Firestore emulator.

**`[spanner](https://cloud.google.com/sdk/gcloud/reference/emulators/spanner)`**:
Manage your local Spanner emulator.

**NOTES**

: These variants are also available:

```
gcloud alpha emulators
```

```
gcloud beta emulators
```
