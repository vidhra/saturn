# gcloud notebooks  |  Google Cloud CLI Documentation

*Source: [https://cloud.google.com/sdk/gcloud/reference/notebooks](https://cloud.google.com/sdk/gcloud/reference/notebooks)*

**NAME**

: **gcloud notebooks - notebooks Command Group**

**SYNOPSIS**

: **`gcloud notebooks` `[GROUP](https://cloud.google.com/sdk/gcloud/reference/notebooks#GROUP)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/notebooks#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Notebooks Command Group.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**GROUPS**

: ``GROUP`` is one of the following:

**`[environments](https://cloud.google.com/sdk/gcloud/reference/notebooks/environments)`**:
Notebooks Environments Command Group.

**`[instances](https://cloud.google.com/sdk/gcloud/reference/notebooks/instances)`**:
Notebooks Instances command group.

**`[locations](https://cloud.google.com/sdk/gcloud/reference/notebooks/locations)`**:
View locations available for a project.

**`[runtimes](https://cloud.google.com/sdk/gcloud/reference/notebooks/runtimes)`**:
Notebooks Runtimes command group.

**NOTES**

: These variants are also available:

```
gcloud alpha notebooks
```

```
gcloud beta notebooks
```
