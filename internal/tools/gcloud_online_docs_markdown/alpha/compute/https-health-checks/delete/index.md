**NAME**

: **gcloud alpha compute https-health-checks delete - delete HTTPS health checks**

**SYNOPSIS**

: **`gcloud alpha compute https-health-checks delete` `[NAME](https://cloud.google.com/sdk/gcloud/reference/alpha/compute/https-health-checks/delete#NAME)` [`[NAME](https://cloud.google.com/sdk/gcloud/reference/alpha/compute/https-health-checks/delete#NAME)` …] [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/alpha/compute/https-health-checks/delete#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `(ALPHA)` `gcloud alpha compute https-health-checks
delete` deletes one or more Compute Engine HTTPS health checks.

**POSITIONAL ARGUMENTS**

: **`NAME` [`NAME` …]**:
Names of the HTTPS health checks to delete.
