**NAME**

: **gcloud alpha compute images vulnerabilities describe-note - describe a Compute Engine image**

**SYNOPSIS**

: **`gcloud alpha compute images vulnerabilities describe-note` `[NOTE_NAME](https://cloud.google.com/sdk/gcloud/reference/alpha/compute/images/vulnerabilities/describe-note#NOTE_NAME)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/alpha/compute/images/vulnerabilities/describe-note#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `(ALPHA)` `gcloud alpha compute images vulnerabilities
describe-note` displays all data associated with a Compute Engine image in
a project.

**POSITIONAL ARGUMENTS**

: **`NOTE_NAME`**:
Name, relative name, or URL of the note.
