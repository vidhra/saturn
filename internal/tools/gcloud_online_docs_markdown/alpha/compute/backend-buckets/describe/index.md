**NAME**

: **gcloud alpha compute backend-buckets describe - describe a backend bucket**

**SYNOPSIS**

: **`gcloud alpha compute backend-buckets describe` `[BACKEND_BUCKET_NAME](https://cloud.google.com/sdk/gcloud/reference/alpha/compute/backend-buckets/describe#BACKEND_BUCKET_NAME)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/alpha/compute/backend-buckets/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `(ALPHA)` `gcloud alpha compute backend-buckets describe`
displays all data associated with a backend bucket in a project.

**POSITIONAL ARGUMENTS**

: **`BACKEND_BUCKET_NAME`**:
Name of the backend bucket to describe.
