**NAME**

: **gcloud alpha bq - interact with and manage resources in Google BigQuery**

**SYNOPSIS**

: **`gcloud alpha bq` `[GROUP](https://cloud.google.com/sdk/gcloud/reference/alpha/bq#GROUP)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/alpha/bq#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: `(ALPHA)` Interact with and manage resources in Google BigQuery.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**GROUPS**

: ``GROUP`` is one of the following:

**`[datasets](https://cloud.google.com/sdk/gcloud/reference/alpha/bq/datasets)`**:
`(ALPHA)` Manage Google BigQuery datasets.

**`[jobs](https://cloud.google.com/sdk/gcloud/reference/alpha/bq/jobs)`**:
`(ALPHA)` Interact with and manage jobs in Google BigQuery.

**`[migration-workflows](https://cloud.google.com/sdk/gcloud/reference/alpha/bq/migration-workflows)`**:
`(ALPHA)` Manage Migration Workflow resources.

**`[tables](https://cloud.google.com/sdk/gcloud/reference/alpha/bq/tables)`**:
`(ALPHA)` Interact with and manage Google BigQuery tables.

**NOTES**

: This command is currently in alpha and might change without notice. If this
command fails with API permission errors despite specifying the correct project,
you might be trying to access an API with an invitation-only early access
allowlist. These variants are also available:

```
gcloud bq
```
