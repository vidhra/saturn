# gcloud ai tensorboards  |  Google Cloud CLI Documentation

*Source: [https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards)*

**NAME**

: **gcloud ai tensorboards - manage Vertex AI Tensorboards**

**SYNOPSIS**

: **`gcloud ai tensorboards` `[COMMAND](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards#COMMAND)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Manage Vertex AI Tensorboards.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**COMMANDS**

: ``COMMAND`` is one of the following:

**`[create](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards/create)`**:
Create a new Vertex AI Tensorboard.

**`[delete](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards/delete)`**:
Delete an existing Vertex AI Tensorboard.

**`[describe](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards/describe)`**:
Gets detailed Tensorboard information about the given Tensorboard id.

**`[list](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards/list)`**:
Lists the Tensorboards of the given project and region.

**`[update](https://cloud.google.com/sdk/gcloud/reference/ai/tensorboards/update)`**:
Update an existing Vertex AI Tensorboard.

**NOTES**

: These variants are also available:

```
gcloud alpha ai tensorboards
```
