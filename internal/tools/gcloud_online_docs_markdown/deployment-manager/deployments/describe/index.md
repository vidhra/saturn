**NAME**

: **gcloud deployment-manager deployments describe - provide information about a deployment**

**SYNOPSIS**

: **`gcloud deployment-manager deployments describe` `[DEPLOYMENT_NAME](https://cloud.google.com/sdk/gcloud/reference/deployment-manager/deployments/describe#DEPLOYMENT_NAME)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/deployment-manager/deployments/describe#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: This command prints out all available details about a deployment.

**EXAMPLES**

: To display information about a deployment, run:

```
gcloud deployment-manager deployments describe my-deployment
```

**POSITIONAL ARGUMENTS**

: **`DEPLOYMENT_NAME`**:
Deployment name.
