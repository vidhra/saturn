**NAME**

: **gcloud access-context-manager levels conditions list - list conditions for an access level**

**SYNOPSIS**

: **`gcloud access-context-manager levels conditions list` (`[--level](https://cloud.google.com/sdk/gcloud/reference/access-context-manager/levels/conditions/list#--level)`=`LEVEL` : `[--policy](https://cloud.google.com/sdk/gcloud/reference/access-context-manager/levels/conditions/list#--policy)`=`POLICY`) [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/access-context-manager/levels/conditions/list#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: List conditions for a basic access level.
