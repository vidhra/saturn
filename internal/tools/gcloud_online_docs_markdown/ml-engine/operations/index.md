# gcloud ml-engine operations  |  Google Cloud CLI Documentation

*Source: [https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations](https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations)*

**NAME**

: **gcloud ml-engine operations - manage AI Platform operations**

**SYNOPSIS**

: **`gcloud ml-engine operations` `[COMMAND](https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations#COMMAND)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Manage AI Platform operations.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**COMMANDS**

: ``COMMAND`` is one of the following:

**`[cancel](https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations/cancel)`**:
Cancel an AI Platform operation.

**`[describe](https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations/describe)`**:
Describe an AI Platform operation.

**`[list](https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations/list)`**:
List existing AI Platform jobs.

**`[wait](https://cloud.google.com/sdk/gcloud/reference/ml-engine/operations/wait)`**:
Wait for an AI Platform operation to complete.

**NOTES**

: These variants are also available:

```
gcloud alpha ml-engine operations
```

```
gcloud beta ml-engine operations
```
