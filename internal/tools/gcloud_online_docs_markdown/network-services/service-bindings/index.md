# gcloud network-services service-bindings  |  Google Cloud CLI Documentation

*Source: [https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings](https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings)*

**NAME**

: **gcloud network-services service-bindings - manage Network Services Bindings**

**SYNOPSIS**

: **`gcloud network-services service-bindings` `[COMMAND](https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings#COMMAND)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Manage Network Services Bindings.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**COMMANDS**

: ``COMMAND`` is one of the following:

**`[create](https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings/create)`**:
Create a service binding.

**`[delete](https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings/delete)`**:
Delete service binding.

**`[describe](https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings/describe)`**:
Describe a service binding.

**`[list](https://cloud.google.com/sdk/gcloud/reference/network-services/service-bindings/list)`**:
List service bindings.

**NOTES**

: These variants are also available:

```
gcloud alpha network-services service-bindings
```

```
gcloud beta network-services service-bindings
```
