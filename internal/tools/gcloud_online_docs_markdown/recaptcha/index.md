# gcloud recaptcha  |  Google Cloud CLI Documentation

*Source: [https://cloud.google.com/sdk/gcloud/reference/recaptcha](https://cloud.google.com/sdk/gcloud/reference/recaptcha)*

**NAME**

: **gcloud recaptcha - manage reCAPTCHA Enterprise Keys**

**SYNOPSIS**

: **`gcloud recaptcha` `[GROUP](https://cloud.google.com/sdk/gcloud/reference/recaptcha#GROUP)` [`[GCLOUD_WIDE_FLAG](https://cloud.google.com/sdk/gcloud/reference/recaptcha#GCLOUD-WIDE-FLAGS) …`]**

**DESCRIPTION**

: Commands for managing reCAPTCHA Enterprise Keys.

**GCLOUD WIDE FLAGS**

: These flags are available to all commands: `[--help](https://cloud.google.com/sdk/gcloud/reference#--help)`.
Run `$ [gcloud help](https://cloud.google.com/sdk/gcloud/reference)` for details.

**GROUPS**

: ``GROUP`` is one of the following:

**`[firewall-policies](https://cloud.google.com/sdk/gcloud/reference/recaptcha/firewall-policies)`**:
Managed reCAPTCHA Firewall Policies.

**`[keys](https://cloud.google.com/sdk/gcloud/reference/recaptcha/keys)`**:
Managed reCAPTCHA Keys.

**NOTES**

: This variant is also available:

```
gcloud alpha recaptcha
```
